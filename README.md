# Library Checker Frontend

## Requirements

- go 1.12 ~

## How to Use

```sh
npm install

# access to the API server of judge.yosupo.jp
npm run start

# access to the API server of local (you must launch api server in local)
REACT_APP_API_URL=http://localhost:58080 npm run start
```
