# Library Checker Frontend

## Requirements

- go 1.12 ~

## How to Use

```sh
cd /path/to/library-checker-frontend

go run . # access to apiv1.yosupo.jp:443 (API server of judge.yosupo.jp)
go run . --local # access to localhost:55051 (you must launch api server)
```
