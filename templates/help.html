Tips

Memory Limit is 1G for all problems. Stack size is unlimited.
メモリ制限は全部1G(230 bytes)です スタックサイズは無制限です

We restart a judge server sometimes. If you submit a solution while restarting, this submission keeps WJ forever(We'll fix it someday...). Please rejudge submission if it wait judging around 10 mins.
時々サーバーを再起動します。その時に提出するとWJのまま無限に放置されます(いつか直す…)。10分ぐらいWJが続くようだったらリジャッジしてみてください。それでダメなら想定外の挙動です。
